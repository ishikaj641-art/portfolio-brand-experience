# Ishika Jain — Personal Brand Experience

A unique, memorable personal brand website designed to showcase strategic thinking, pattern recognition, and entrepreneurial mindset.

## 🎯 What Makes This Different

This isn't a traditional portfolio. Instead of listing achievements, it:

- **Shows how you think** through storytelling and observations
- **Demonstrates strategic depth** by revealing insights others miss
- **Positions you as a leader** through brand and innovation thinking
- **Creates memorable moments** that recruiters and investors won't forget

## 📋 Structure

### Pages & Sections

1. **How I Think** - Your mental models and approach
2. **What I've Built** - Projects told through the lens of learning
3. **What I Notice** - Five key observations about markets and human behavior
4. **What Excites Me** - Forward-thinking interests and problems you're obsessed with
5. **Let's Talk** - Clear CTA and contact methods

## 🛠 Files

- `index.html` - Main structure and content
- `styles.css` - Design system and responsive layout
- `script.js` - Smooth scroll, animations, and interactions
- `README.md` - This file

## 🚀 Deployment

### GitHub Pages
1. Push to your GitHub repository
2. Go to Settings → Pages
3. Select "Deploy from a branch" → main
4. Your site will be live at `https://yourusername.github.io/portfolio-brand-experience`

### Netlify (Recommended)
1. Connect your repository to Netlify
2. Auto-deploys on every push
3. Get a custom domain

## ✨ Customization

### Update Contact Information
In `index.html`, update:
```html
<a href="mailto:your.email@example.com" class="btn btn-primary">Email me</a>
<a href="https://linkedin.com/in/yourprofile" class="btn btn-secondary" target="_blank">LinkedIn</a>
```

### Change Colors
In `styles.css`, modify the CSS variables in `:root`:
```css
--color-accent: #e74c3c; /* Change this color */
--color-gray-dark: #1a1a1a;
--color-gray-light: #f5f5f5;
```

### Update Content
Edit text directly in `index.html`. All copy is structured for easy updates.

## 🎨 Design Philosophy

- **Typography-First** - Serif fonts for impact, sans-serif for readability
- **Minimal & Spacious** - Breathing room between ideas
- **Intentional Color** - Accent color draws attention to key insights
- **Card-Based** - Easy scanning and visual hierarchy
- **Dark Mode Section** - Contrast for the "What I've Built" section

## 💡 Tips for Success

3. **Keep copy fresh** - revisit and update annually
